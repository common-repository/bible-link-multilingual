This lightweight plugin makes Bible references on your website interactive and supports multiple languages.

== Description ==
This lightweight plugin makes Bible references on your website interactive, showing a preview when the visitor clicks or moves the mouse cursor over the references. Up to three verses are shown, followed by a 'Read more' link. 

You can choose from 20+ languages and 50+ bible versions. Check the list of available languages/bibles [here](https://bible-link.com/languages-and-translations/).

https://www.youtube.com/watch?v=c-i_3ALvw-0

[bible-link.com](https://bible-link.com/)

== Installation ==
In WordPress:
1. Navigate in the wordpress admin to plugins->add new.
2. Search for 'BibleLink Multilingual' by GlobalRize.
3. Install and activate the plugin.
Note: by installing the plugin you [agree to the terms](https://bible-link.com/terms-of-use/) for the use of the plugin.
